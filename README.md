## üìå About the Project

**AsbYab** was a web application developed in 2022 for horse enthusiasts and riders. It allowed users to search, view, and manage information about horses, riding clubs, and competitions. The blog section was powered by **WordPress** to provide news, tips, and articles about horses and equestrian events.  

This project demonstrates my ability to build **functional, responsive, and user-friendly web applications** using **PHP, HTML, CSS, Bootstrap, JavaScript, and WordPress**.

---

## üéØ Features

- **Advanced Horse Search:** Filter horses by breed, age, skill level, and location  
- **Club & Trainer Management:** Add and view authorized clubs and trainers  
- **Competition Management:** View schedules and results  
- **Responsive Design:** Works on desktop and mobile using Bootstrap  
- **User Accounts:** Registration, login, and profile management  
- **Blog Section:** WordPress-powered blog for news and tips
